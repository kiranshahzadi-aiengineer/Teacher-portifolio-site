## 🌟 Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Interactive Elements** - Animated skill bars, smooth scrolling navigation
- **Contact Form** - Direct email integration for easy communication
- **Professional Sections**:
  - Hero section with key statistics
  - Comprehensive professional profile
  - Detailed work experience (9+ years)
  - Educational credentials
  - Skills and expertise showcase
  - Contact information

## 📝 How to Enable GitHub Pages

1. Go to your repository: https://github.com/kiranshahzadi-aiengineer/Teacher-portifolio-site
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** section (left sidebar)
4. Under **Source**, select **main** branch
5. Click **Save**
6. Your website will be live in a few minutes at the URL above

## 🔧 Customization

### Change Colors
Edit `style.css` and modify the CSS variables:
```css
:root {
    --primary-color: #2563eb;    /* Main blue color */
    --secondary-color: #1e40af;  /* Darker blue */
    --accent-color: #60a5fa;     /* Light blue */
}
```

### Update Content
Edit `index.html` to update:
- Personal information
- Experience details
- Skills and qualifications
- Contact information

### Add Your Photo
Insert an image in the hero section:
```html
<img src="your-photo.jpg" alt="Kiran Shahzadi" class="hero-image">
```
